This is not the web portfolio you are looking for

I've always been in Enterprise IT. But here's some little web learning projects I've dropped on the free clouds.

Big screens

  • PhraseOmatic. A Java servlet example based on HeadFirst Java. Runs on Google Cloud App Engine with Cloud Storage buckets for nonsense management.

  • City-Weather picker - Python - Flask on Goo App Engine
    This is a current weather picker based on Tristan Ganry's Medium article, grabbing weather info from OpenWeather API. As in the word salad app above, I'm using a Google Cloud bucket for city names list management.

  • City-Weather again - Node/Express on Heroku
    Same API. This is based on Brandon Morelli's blogtorial in his codeburst offering. I tweaked it for Heroku. I added Google Places API autocomplete feature to the City field, sadly limited to USA. You can still type Timbuktu and Whakapirau. TODO: Autocomplete rest of the World.

  • Wild Rydes! An AWS tutorial app - It's Ubercorns. Register and call for a RYDE! Service defaults to Seattle but I called one to Boise, no Prob. Scroll waayyy... down to the footer to call your Unicorn for a Ryde.

    The AWS paint-by-numbers tutorial wires up provided resources including DynamoDB, a REST API, Lambda functions and psychedelic page art.

  • App Engine Static sites Playbox
    GAE Python webapp2 framework serving static web sites. Some silly pages messing with Javascript and web tools like Material Design Light.

  • Song List
    Hello Rails demo on heroku. Add a song!

  • Rails Contacts Multi-User
    Rails login/register/forgot demo using Devise on heroku

Little screens

    The original demo I found is hard to read, so I made the compass needle pointers friendlier with CSS shapes.

  • Behold the Tartanator
    Mobile web app from the Head First Mobile Web book. Lookout, I think it's got some PHP in there, handled by Heroku.
