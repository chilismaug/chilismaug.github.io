Big screens

  • City-Weather picker - Flask on App Engine
    Playing on Google Cloud, using Python's Flask framework. This is a current weather picker based on Tristan Ganry's Medium article, grabbing weather info from an API - I added a JSON data file for the city picklist and merged request/response to one page.

  • App Engine Static sites Playbox
    GAE Python webapp2 framework serving static web sites. Some silly pages messing with Javascript and web tools like Material Design Light.

  • Song List
    Hello Rails demo on heroku. Add a song!

  • Wild Rydes! An AWS tutorial app where you follow the steps to wire up AWS services including DynamoDB, REST API and Lambda. It's Ubercorns. You can register and call for a RYDE! Service defaults to Seattle but I called one to Boise, no Prob. I added a footer link (scroll waayyy... down) to call your Unicorn for a Ride. All the psychedelic page art and Lambda code was provided, FYI.

  • Rails Contacts Multi-User
    Rails login/register/forgot demo using Devise on heroku

    This is from partway through Dane Cameron's book. No back-end yet.
