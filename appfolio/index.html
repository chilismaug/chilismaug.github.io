This is not the portfolio

Since I've always been in "Enterprise IT" doing B2B stuff, there's no street-facing art to show from work. But here's some things that I've munged together based on tutorial/sample code and dropped on the free clouds.

Big screens

  • PhraseOmatic. A Java servlet example that started out from HeadFirst Java, with some enhancements, like a style sheet, and data loaded from text files rather than inline. This is deployed to Google Cloud Platform's App Engine. I just figured out how to get the files to be streamed from Google Cloud Storage buckets, so I can tweak my nonsense without recompiling. Content Management at last.

  • City-Weather picker - Flask on App Engine
    Playing on Google Cloud, using Python's Flask framework. This is a current weather picker based on Tristan Ganry's Medium article, grabbing weather info from an API - I added a JSON data file for the city picklist, housed in a Goo Cloud Storage bucket, so I can change City data without touching code. On the UI side, I merged request/response views to act more "single-pagey".

  • City-Weather again via Node/Express on Heroku
    Same API from a different way. This is based on Brandon Morelli's blogtorial in his codeburst offering, with a few tweaks to get it on Heroku. I got a Google Places API autocomplete feature working on the City field.

  • Wild Rydes! An AWS tutorial app - It's Ubercorns. You can register and call for a RYDE! Service defaults to Seattle but I called one to Boise, no Prob. I added a footer link (scroll waayyy... down) to call your Unicorn for a Ride.

    To build this you follow the steps to wire up AWS services including DynamoDB, REST API and Lambda. All the psychedelic page art and Lambda code was provided, FYI. I lapsed my AWS subs but the app still lives.

  • App Engine Static sites Playbox
    GAE Python webapp2 framework serving static web sites. Some silly pages messing with Javascript and web tools like Material Design Light.

  • Song List
    Hello Rails demo on heroku. Add a song!

  • Rails Contacts Multi-User
    Rails login/register/forgot demo using Devise on heroku

    The original demo is kind of hard to read, so I made the compass needle pointers friendlier with CSS shapes.
    Make sure it's via https and you "Allow Location Access" spying in your desktop.
